Debian Guest Installation

Installing Debian Gnu/Linux​

  • VirtualBox Name: DEB00-2

START Install​

  • Select START

Debian Installer Menu​

  • Select Graphical Install

Select A Language​

  • Select English

Select Your Location​

  • Other (Or else)

  • Asia (Or else)

  • Indonesia (Or else)

Configure Locales​

  • Select Australia: en_AU.UTF-8
    • Australia is an English-speaking country with a 24-hour clock system.

Configure Keyboard​

  • Select American English

Configure Network (e.g. cbkadal)​

  • Hostname: cbkadal

  • Domain Name: BLANK

  • ROOT password: cbkadal

Configure The Clock​

  • Select Western (Sumatera, Jakarta, West/Central Kalimantan)

Partition Disk​

  • Partitioning Method:
    • Select Guided -- use entire disk

  • Scan Disk to Partitioning

  • Partitioning Scheme:
    • Select All files in one partition

  • Select Finish Partitioning

  • Write to Disk: Yes

Install Base System​

Configure The Package Manager​

  • Scan Extra Media:
    • Select NO

  • Debian Archive Mirror Country:
    • Select Indonesia

  • Debian Archive Mirror:
    • Select deb.debian.org

  • HTTP Proxy: blank

Configuring Popularity Contest​

  • The default is: NO; but there is nothing wrong if you choose YES!

Software Selection​

OptionSelection
Debian Desktop EnvironmentNO
... GNOMENO
SSH serverYES
standard system utilityYES

Install The GRUB Boot Loader​

  • Install the GRUB Boot Loader to your primary drive:
    • Select YES

  • Install the GRUB Boot Loader:
    • Select /dev/sda

Finish Installation​

Reboot​

  • Select: Debian GNU/Linux

Test​

Test Login​

  • login: YOUR ACCOUNT
  • password: YOUR PASSWORD

Substitute "ROOT"​

  • Enter ROOT password
su -

Test Network/Update (as ROOT)​

apt-get update
apt-get dist-upgrade -y

poweroff Shutdown​

poweroff
